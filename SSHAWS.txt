Launch 2 AWS instances

Since you already have the .pem keys for both EC2 instances, the process is slightly different because AWS Linux instances (like Amazon Linux or Ubuntu) have password authentication disabled by default.
Here is how to set this up for both scenarios:

ssh-keygen -t Ed25519 -b 4096
In WSL, your Windows files are mounted under the /mnt/ directory.

 mv /mnt/c/Users/user/Downloads/MN1.pem ~/.ssh/
 ls -l ~/.ssh/
